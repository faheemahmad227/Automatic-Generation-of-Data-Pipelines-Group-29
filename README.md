# Automatic Data Pipeline Generator

A web application that automatically generates executable Python data pipelines from natural language descriptions. Built for the Data-driven Engineering course at Paderborn University.

## Features

- **Natural Language Pipeline Generation**: Describe your pipeline in plain English, and the system generates production-ready Python code
- **AI-Powered Analysis**: Automatic schema detection, data quality assessment, and transformation recommendations
- **V-Model Support**: Full support for all nine V-Model phases (Requirements through Acceptance Testing)
- **Complete Pipeline Package**: Generates Python code, YAML configuration, and test suites
- **Real-time Monitoring**: Dashboard for tracking pipeline execution and metrics
- **Multiple Data Formats**: Supports CSV, JSON, Excel, and Parquet files

### Prerequisites

- Python 3.10 or higher
- pip (Python package manager)
- 16GB RAM recommended
- (Optional) OpenAI API key or Uni Paderborn AI Gateway access
